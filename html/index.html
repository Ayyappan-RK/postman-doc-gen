POST Create Group Set
https://cpd-service.dev.apps.scholastic.tech/classes/84200/group-sets?num-groups=2

Acceptable values for groupType and tieBreaker are - TEACHER_SELECTED, GRL, LEXILE, SKILL, INTEREST

For teacher selected, tieBreaker must be null.

HEADERS
Content-Type
application/json
PARAMS
num-groups
2
